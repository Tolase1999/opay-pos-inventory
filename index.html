⏱ How Old Stock works: Each SN's age is calculated from its original receipt date (when it was first added to inventory). 📢 Broadcast Message
→ Send scrolling ticker messages to all officers

    

      
📢
Broadcast to Officers
Type a message — it appears as a live scrolling ticker on every officer's screen

      

        

          ⚠️ One-time Supabase setup: Copy and run this SQL in your Supabase dashboard → SQL Editor to create the broadcast table:
          
CREATE TABLE IF NOT EXISTS broadcast_messages (
  id BIGSERIAL PRIMARY KEY,
  message TEXT NOT NULL,
  sender TEXT NOT NULL,
  role TEXT NOT NULL,
  is_active BOOLEAN DEFAULT true,
  created_at TIMESTAMPTZ DEFAULT NOW()
);
ALTER TABLE broadcast_messages ENABLE ROW LEVEL SECURITY;
CREATE POLICY "anon_all" ON broadcast_messages FOR ALL USING (true) WITH CHECK (true);

-- Enable Realtime (required for instant ticker updates)
ALTER PUBLICATION supabase_realtime ADD TABLE broadcast_messages;
